## 📋 Sobre o projeto

O **ModaOra** é um blog de moda desenvolvido como projeto de estudos, focado em layout, tipografia e responsividade. O site apresenta conteúdo editorial de moda com uma interface limpa e elegante e adaptação perfeita para qualquer tamanho de tela.

---

## ✨ Funcionalidades

- 📱 **Layout responsivo** — adapta-se a celulares, tablets e desktops
- 🗂️ **Estrutura de blog** — seções de artigos e posts organizados
- 🎨 **Design elegante** — paleta de cores e tipografia pensadas para moda

---

## 🛠️ Tecnologias utilizadas

| Tecnologia | Uso |
